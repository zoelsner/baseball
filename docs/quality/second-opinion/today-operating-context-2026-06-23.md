# Second Opinion: Today Operating Context

## Context

Ramp's Agentic CX role emphasizes applied AI operations, human/AI workflows,
operator empathy, quality standards, evals, and reliable automation. Sandlot's
Today screen should show those product principles through the fantasy-baseball
workflow.

## Proposed Slice

- Move current matchup status to the top of Today.
- Keep Hot Swaps directly below it as the recommended next action.
- Keep Attention Queue below Hot Swaps as secondary monitoring.
- Make Hot Swaps copy reference matchup context so the page reads as
  observe -> decide -> safely block or hand off, not as unrelated widgets.
- Keep all Fantrax, Zo, add/drop, and trade writes disabled.

## Review Prompt

Pressure-test whether this approach makes the app read like an agentic
operations system instead of a generic fantasy app. Call out must-fix concerns,
product/story improvements, and verification requirements.
